Question-1: What are the different ways to select an element in the DOM?

By ID (getElementById), By Class Name, By Tag Name, By Query Selector,By Query Selector All

Question-2: What is the difference between innerHTML, innerText, and textContent ?

InnerHTML for HTML content, innerText for visible text, and textContent for all text including hidden

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for its child elements using event bubbling.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event starts from the target element and propagates up to its ancestors in the DOM.

Question-5: How do you create, add, and remove elements using JavaScript?

let el = document.createElement("div"); document.body.appendChild(el); document.body.removeChild(el);
